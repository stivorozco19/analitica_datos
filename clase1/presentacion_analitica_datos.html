Entendiendo Cada Nivel de la Pirámide

📦

DATOS - Los Hechos Crudos

¿Qué son? Información sin procesar, números sueltos, texto sin contexto.
Características: Sin significado por sí solos, necesitan ser organizados.
Ejemplos: 25°C, $150,000, "Juan", 2024-03-15

📈

INFORMACIÓN - Datos Procesados

¿Qué es? Datos organizados, calculados y con contexto.
Características: Responde preguntas básicas (qué, cuándo, dónde).
Ejemplos: "Temperatura promedio: 27.7°C", "Ventas febrero: $45M"

🧠

CONOCIMIENTO - Insights Accionables

¿Qué es? Interpretación de información que genera valor.
Características: Responde "por qué" y sugiere "qué hacer".
Ejemplos: "Subir temp. → +20% ventas helados → aumentar stock"

Casos Empresariales: Cómo se Hace Analítica Real

Vamos más allá del "qué" para entender el "cómo"

🎬

Netflix: El Sistema de Recomendaciones

Problema: Con 15,000 títulos, ¿cómo evitar que usuarios se abrumen y cancelen?

📊 Los Datos que Recolectan:

  • • ¿Qué miras? (título, género, duración)
  • • ¿Cuándo lo miras? (hora, día, dispositivo)
  • • ¿Cómo lo miras? (pausas, rebobinados, abandono)
  • • ¿Qué buscas? (términos, navegación)
  • • ¿Cómo calificas? (thumbs up/down)
  • • ¿Quién eres? (perfil, ubicación, suscripción)

🔍 Cómo Procesan la Información:

Filtrado Colaborativo

"Usuarios como tú también vieron..."

Análisis de Contenido

"Te gustó X, esto es similar..."

Algoritmos Híbridos

Combina múltiples señales

🧠 El Conocimiento que Generan:

💡 Insights Operativos:

  • • 70% abandonan si no encuentran algo en 90 segundos
  • • Usuarios nocturnos prefieren comedias
  • • Fin de semana = más tiempo para series largas

🎯 Decisiones de Negocio:

  • • Invertir $15B en contenido original
  • • Crear diferentes portadas por región
  • • Optimizar horas de lanzamiento global
🏆 RESULTADO: $1B+ anuales ahorrados en retención de clientes
🏦

Bancolombia: Credit Scoring Inteligente

Problema: ¿A quién prestar dinero sin quebrar por morosidad?

📊 Fuentes de Datos Innovadoras:

Datos Tradicionales:

  • • Historial crediticio (Datacrédito)
  • • Ingresos declarados
  • • Patrimonio y garantías

Datos Alternativos:

  • • Comportamiento transaccional
  • • Geolocalización de pagos
  • • Patrones de ahorro

⚙️ Cómo Construyen el Modelo:

1. Segmentación

Empleados, independientes, pensionados

2. Variables

300+ variables por cliente

3. Machine Learning

Random Forest + XGBoost

4. Score Final

0-1000 puntos

🎯 Insights que Descubrieron:

🔍 Patrones Inesperados:

  • • Clientes que pagan servicios los viernes son más confiables
  • • Uso de cajeros de otros bancos indica estabilidad financiera
  • • Pequeños ahorros constantes > grandes depósitos esporádicos

💼 Decisiones Implementadas:

  • • Preaprobación automática para score >750
  • • Tasas dinámicas según comportamiento
  • • Alertas tempranas de deterioro crediticio
🏆 RESULTADO: 30% reducción en morosidad + 25% más créditos aprobados
