Principio fundamental: "Basura entra → Basura sale"

⚡ ¿Por Qué la Limpieza es CRÍTICA?

🗑️ "Basura Entra → Basura Sale"

Sin datos limpios, todo el análisis posterior será incorrecto

❌ Missing Values

Problema: Cálculos de ventas incorrectos

Ejemplo: "El promedio mensual es $2M" (pero faltaban 10 días de datos)

❌ Duplicados

Problema: KPIs inflados artificialmente

Ejemplo: "Vendimos $5M" (pero contamos la misma venta 3 veces)

❌ Tipos Incorrectos

Problema: Gráficos que no se pueden crear

Ejemplo: Fechas como texto impiden análisis temporal

❌ Outliers sin Detectar

Problema: Tendencias distorsionadas

Ejemplo: Un error de $999,999 en una venta de $99

🧹 Limpieza de Datos con Pandas

🔍 Detección de Missing Values

# Detectar valores faltantes df.info() df.isnull().sum() df.describe() # Visualizar patrones df.isnull().sum().plot(kind='bar')

🧽 Manejo de Missing Values

# Diferentes estrategias df.fillna(method='ffill') # Forward fill df.fillna(df.mean()) # Imputación con media df.dropna() # Eliminar filas df.fillna(0) # Llenar con cero

🔄 Eliminación de Duplicados

# Detectar duplicados df.duplicated().sum() df[df.duplicated()] # Eliminar duplicados df.drop_duplicates() df.drop_duplicates(subset=['column'])

📊 Cambio de Tipos de Datos

# Cambiar tipos df['fecha'] = pd.to_datetime(df['fecha']) df['precio'] = df['precio'].astype('float') df['categoria'] = df['categoria'].astype('category')

🔧 Transformaciones de Datos

➕ Creación de Nuevas Columnas

# Nuevas columnas calculadas df['total'] = df['precio'] * df['cantidad'] df['categoria_precio'] = df['precio'].apply( lambda x: 'Alto' if x > 100 else 'Bajo' ) df['mes'] = df['fecha'].dt.month

🧹 Limpieza de Strings

# Limpiar texto df['producto'] = df['producto'].str.strip() df['producto'] = df['producto'].str.upper() df['precio_str'] = df['precio_str'].str.replace('$', '') df['precio_str'] = df['precio_str'].str.replace(',', '')

📅 Manejo de Fechas

# Trabajar con fechas df['fecha'] = pd.to_datetime(df['fecha'], format='%d/%m/%Y') df['año'] = df['fecha'].dt.year df['dia_semana'] = df['fecha'].dt.day_name()

🎯 Filtrado y Selección

# Filtrar datos ventas_altas = df[df['precio'] > 100] productos_tech = df[df['categoria'] == 'Tecnología'] ventas_2024 = df[df['fecha'] >= '2024-01-01']

📐 Principios de Tidy Data

🎯 ¿Qué son los "Tidy Data"?

Datos organizados de forma consistente y fácil de analizar

1️⃣ Cada Variable = Una Columna

✅ Correcto: columnas separadas para 'precio' y 'cantidad'

❌ Incorrecto: 'precio_cantidad' con valores mixtos

2️⃣ Cada Observación = Una Fila

✅ Correcto: cada venta en una fila diferente

❌ Incorrecto: múltiples ventas en una sola fila

3️⃣ Cada Tipo = Una Tabla

✅ Correcto: tabla de ventas, tabla de productos

❌ Incorrecto: mezclar ventas con inventario

4️⃣ Un Valor = Una Celda

✅ Correcto: una fecha por celda

❌ Incorrecto: "2024-01-15 hasta 2024-01-20"

🔢 Introducción a GroupBy

GroupBy: La base para KPIs de negocio

Ahora que tenemos datos limpios, podemos hacer análisis que Sheets no maneja

📊 Análisis Por Categoría

# Ventas promedio por categoría df.groupby('categoria')['precio'].mean() # Total de ventas por categoría df.groupby('categoria')['total'].sum()

🌍 Análisis Por Región

# Ventas por región y categoría df.groupby(['region', 'categoria']).agg({ 'precio': 'mean', 'cantidad': 'sum', 'total': 'sum' })

📅 Análisis Temporal

# Ventas por mes df.groupby(df['fecha'].dt.month)['total'].sum() # Tendencia mensual df.set_index('fecha').resample('M')['total'].sum()

🎯 KPIs Empresariales

# Ticket promedio por cliente df.groupby('cliente_id')['total'].mean() # Productos más vendidos df.groupby('producto')['cantidad'].sum().sort_values(ascending=False)
